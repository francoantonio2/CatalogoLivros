Catálogo de Livros em Java

Descrição

Este projeto foi desenvolvido em Java com o objetivo de aplicar conceitos de Programação Orientada a Objetos (POO).

O sistema realiza o cadastro de livros utilizando um array de objetos e exibe apenas os livros cujo título contém uma palavra-chave específica ("Java").


Funcionalidades

Cadastro de livros em um array
Filtro de livros por palavra-chave no título
Uso do método `contains()` da classe String
Exibição das informações no console


Conceitos aplicados

Classe e objeto
Construtor
Array de objetos
Laço de repetição (`for`)
Manipulação de Strings
Encapsulamento básico

Estrutura do projeto

src/

      └── edu/

Via terminal:


javac br/edu/catalogo/*.java
java br.edu.catalogo.Principal
